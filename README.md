# Trade Document Intelligence

AI-powered extraction of structured data from commercial invoices
for Spanish-language markets (Latin America & Spain).

## What this does
Extracts customs-relevant fields from commercial invoices using
the Anthropic Claude API:
- Shipper / Consignee
- HS Codes
- Declared value and currency
- Country of origin
- Net weight / Gross weight

## Current status
- Sprint 2 complete — extraction pipeline working
- Accuracy on first document: 10/10 fields (100%)
